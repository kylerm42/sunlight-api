# Sunlight Api

Gem for interacting with the various Sunlight Foundation APIs

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sunlight-api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sunlight-api
